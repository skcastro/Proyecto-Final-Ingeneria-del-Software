Pasos para correr el proyecto:

1. abrir PekselG5.sln
2. Ir a appsettings.json y cambiar el server en el string de conexion
3. Abrir el sql server manager
4. En Visual Ir a Tools/Herramientas -> NuGet Package Manager -> Package Manager Console / Consola de Manejador de Paquetes
5. En la consola escribir update-database
Con eso se les crea la base de datos en el server de sql y ya con eso pueden ver el app
6. Corran el app
7. Registring su usuario
8. Logearse con el usuario
9. ya pueden usar el app, vean que si no se loguean no pueden hacer nada.
